# 业务类别三级下拉框配置

> 用于系统中“大类 → 中类 → 小类”三级联动下拉框。

## 下拉框规则

1. **一级下拉框：大类**
2. **二级下拉框：中类**，根据所选大类动态加载
3. **三级下拉框：小类**，根据所选中类动态加载
4. 原表中的 `/` 表示**无下一级分类**，系统中不要将 `/` 作为可选项
5. 当大类无中类时，选择大类后即结束分类，隐藏或禁用中类、小类下拉框
6. 当中类无小类时，选择中类后即结束分类，隐藏或禁用小类下拉框
7. 原表中“用工管理”重复出现，系统下拉框中合并为一个大类

## 前端交互建议

- 初始状态：仅“大类”下拉框可选
- 选择大类后：
  - 有中类：加载中类选项
  - 无中类：直接完成分类
- 选择中类后：
  - 有小类：加载小类选项
  - 无小类：直接完成分类
- 用户修改上一级选项时，自动清空所有下级已选值
- 最终保存时建议同时保存：
  - `largeCategory`：大类
  - `mediumCategory`：中类，可为空
  - `smallCategory`：小类，可为空
