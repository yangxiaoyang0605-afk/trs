# 日照银行基础 UI 设计规范（AI 读取版）

> **用途**：本文件用于 AI、设计系统、前端开发与界面生成工具直接读取。
>
> **数据原则**：所有颜色、字号、间距、圆角、阴影、栅格、断点和布局尺寸均依据原始 7 页 UI 规范整理。不得自行改数、取整、近似、替换或“优化”。
>
> **单位规范化说明**：原稿中个别字号示例存在单位字形缺失（例如 `32x`）；本文件按其对应 token 与上下文统一规范为 `32px`，**数值保持不变**。

## 0. AI 执行硬规则

1. 只能使用本文件明确给出的数值和颜色。
2. 不得把相近值互换。例如：
   - 功能蓝默认色为 `#3D71FF`；数据图表配色一默认色为 `#3C71FF`，两者不同。
   - 1920px、1440px 各场景的外边距不同，不得混用。
3. 布局必须先判断场景，再选用对应模板：
   - 非首页三栏（1920px）
   - 非首页两栏（1920px）
   - 首页-登录后（1440px）
   - 其他页面两栏/三栏（1440px）
4. 首页-登录前、首页-登录后不属于通用适配规则，需使用单独适配设计稿；本文件仅记录原规范中明确给出的“首页-登录后 1440px”尺寸。
5. 原规范同时给出“栅格系统页面边距”和“具体布局模板外边距”。它们是不同层级的规则，不得互相替代：
   - 栅格系统页面边距：1920px → 136px；1440px → 48px。
   - 具体布局模板外边距：见第 7 节各模板尺寸。

## 2.2 字号总规则

- 最小可识别文字尺寸：`12px`。
- 字体层级差距：`2-4px`。
- 产品中的字体层级建议不超过 `5` 种。

- 多行与单行文本行高均采用 `1.5` 倍行高。

## 2.5 数字字体

### 应用场景

数字字体主要用于页面特殊场景，以展示重要数字信息，例如：

- 首页资产负债总览总金额

3. 当组件极限展示涉及 `16`、`14`、`12` 号等 Sccba Type 数字字体时，以组件样式为准实施开发。

# 3. 通用投影

## 3.1 阴影层级

系统分为低、中、高 `3` 个 UI 层级。对象离地面越远，阴影越大，模糊值越高。

### 低阴影

- 用途：默认 / hover 状态，例如表格和树拖动。
- CSS：

```css
box-shadow: 3px 5px 8px 0px rgba(0, 0, 0, 0.02);
```

### 中阴影

- 用途：悬浮卡片 / 提示，例如下拉菜单、气泡确认框、选择器等。
- CSS：

```css
box-shadow: 0px 4px 12px 0px rgba(0, 0, 0, 0.08);
```

### 高阴影

- 用途：强调 / 抽屉，例如全局提示、消息通知等。
- CSS：

```css
box-shadow: 0px 17px 24px 0px rgba(0, 0, 0, 0.06);
```

# 4. 通用间距

## 4.1 间距类型

- 水平间距：横向相邻组件 / 卡片的间距。
- 垂直间距：纵向相邻组件 / 卡片的间距。
- 页面边距：页面或卡片内信息与边缘的间距。

## 4.2 水平间距

- 采用 `4px` 倍数规律。
- 原规范明确示例：`8px`、`16px`、`20px`。
- 内容相关性越高，间距越小。
- 默认卡片间距：`16px`，不论卡片宽高。

## 4.3 垂直间距

- 采用 `4px` 倍数规律。
- 原规范明确示例：`8px`、`16px`、`20px`。
- 内容相关性越高，间距越小。
- 默认卡片间距：`16px`，不论卡片宽高。

## 4.4 页面边距

- 大部分页面内信息距离页面上下左右边缘：`24px`。
- 出现页面多层级时，以 `4px` 的倍数增减调整；具体边距以实际页面为准。
- 原规范的页面嵌套示意值为 `20px`；该值是示例，不应覆盖所有嵌套场景。

# 8. AI 生成界面时的决策顺序

AI 必须按以下顺序执行：

1. 判断画布宽度：1024 / 1440 / 1920 / >1920。
2. 判断页面类型：首页或非首页。
3. 判断布局类型：两栏或三栏。
4. 优先匹配第 7 节的精确模板。
5. 再应用第 6 节栅格系统。
6. 再应用第 4 节间距和第 5 节圆角。
7. 最后应用颜色、字体和阴影。
8. 任何未在规范中定义的值，不得从相近 token 推导；应标记为“未定义，需要组件级规范”。

---

# 9. 禁止事项

- 禁止将 `#3D71FF` 与 `#3C71FF` 合并。
- 禁止将 `H18` 的实际字号改为 18px；原规范实际字号是 `16px`。
- 禁止将 1440px 首页外边距 `14px` 用于其他页面。
- 禁止将 1440px 其他页面外边距 `10px` 用于首页-登录后。
- 禁止将 1920px 具体布局外边距 `24px` 与栅格系统页面边距 `136px` 混为同一变量。
- 禁止把默认卡片间距 `16px` 改为 12px、20px 或其他值。
- 禁止把默认卡片圆角 `16px` 改为 12px；12px 用于大部分卡片内组件和嵌套内层卡片。
- 禁止给主题渐变自行指定固定角度，因为原规范未明确角度。
